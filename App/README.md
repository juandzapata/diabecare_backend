# diabecare_backend

Descripción breve del proyecto.

## Requisitos

- Python 3.8+
- FastAPI
- SQLAlchemy

## Instalación

Primero, clona el repositorio a tu máquina local:

```bash
git clone https://github.com/usuario/proyecto.git
cd proyecto
pip install -r requirements.txt
```
## Correr el proyecto
```bash
uvicorn main:app --reload
```
